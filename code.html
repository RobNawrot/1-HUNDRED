                        // Get the Values from the Page. (Controller Function)
                        function getValues() {
                            
                            // Get values from the page.
                            let startValue = document.getElementById("startValue").value;
                            let endValue = document.getElementById("endValue").value;

                            // Attempt to cast the string values into integer values.
                            startValue = parseInt(startValue);
                            endValue = parseInt(endValue);

                            // Input given for start and end values must be validated.
                            if (Number.isInteger(startValue) && Number.isInteger(endValue) &&
                            startValue >= 0 && startValue < 100 && endValue > 0 && endValue <= 100) {
                                // Call generateNumbers().
                                let numbers = generateNumbers(startValue, endValue);
                                // Call displayNumbers().
                                displayNumbers(numbers);  
                            }
                            else {
                                alert("ERROR - You must enter integer values within the range of 0 - 100.");
                            }

                        } 

                        // Generate numbers from the start value to the end value. (Logic Function)
                        function generateNumbers(startValue, endValue) {

                            let numbers = [];

                            // We want to get all numbers from start to end.
                            // This loop will iterate until i = endValue.
                            for (let i = startValue; i <= endValue; i++) {
                                numbers.push(i);
                            }

                            return numbers;

                        }

                        // Display the numbers and mark EVEN numbers BOLD. (Display Function)
                        function displayNumbers(numbers) {
                            
                            // General string used for generating HTML table on the app page.
                            let templateRows = "";
                            
                            // Loops through the 'numbers' array of integer values and builds the HTML
                            // string.
                            for (let i = 0; i < numbers.length; i++) {
                                
                                let className = "even";
                                let number = numbers[i];

                                // Checking whether the current number in the array is odd or even.
                                // If even, the number will be BOLDED.
                                if (number % 2 == 0) {
                                    className = "even";
                                }
                                else {
                                    className = "odd";
                                }

                                // Concatenates an HTML table row code onto the general string. 
                                templateRows += `<tr><td class="${className}" >${number}</td></tr>
                                
                            }

                            // Inserts the concatenated string into the 'tbody' HTML element.
                            document.getElementById("results").innerHTML = templateRows;

                        }

The Code is structured in three functions.

getValues

getValues is a function that retrieves the starting and ending integer values from the input fields. This determines where to start and stop counting. It takes no parameters and returns nothing.

This function ensures that the input remains consistent by only accepting integer values between the 0 - 100 range.

First, an attempt to parse the input into integers is made by converting the string literals returned to startValue and endValue. After parsing, the values are validated as a final check that they are in fact integers.

If the values are valid, a call to the generateNumbers function is made, followed by a call to displayNumbers. If the values are invalid, then an alert is sent to the user notifying them of the error.

generateNumbers

generateNumbers is a function that populates an array with a subset of integer values determined by a given starting point (startValue) and ending point (endValue).

This function takes two integer parameters and returns an array of integers.

displayNumbers

displayNumbers is a function responsible for displaying the resulting integer count on the web page.

It takes a single integer array parameter and returns nothing.

This function contains a single for-loop that iterates over the integer array passed to it; concatenating a new HTML table row element for each integer part of the array.

Additionally, this function displays each even number in BOLD for greater readability.
